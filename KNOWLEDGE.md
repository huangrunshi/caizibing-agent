## 一、产品介绍

### 什么是菜籽饼？
菜籽饼是油菜籽榨油后的副产品，富含氮、磷、钾及多种微量元素，是优质天然有机肥料。

### 营养成分（参考值）
| 成分 | 含量 |
|------|------|
| 氮(N) | 4.6% |
| 磷(P₂O₅) | 2.5% |
| 钾(K₂O) | 1.4% |
| 有机质 | ≥75% |

### 产品规格
- **形态**：压榨饼块 / 粉碎粉末
- **规格**：散称（按斤）、袋装（5斤/10斤/20斤）、批量（100斤起）
- **来源**：自家小作坊压榨，新鲜无添加
- **卖点**：纯天然、无化学添加剂、本地原料、现榨现售

## 二、发酵方法（重要！）

### 为什么要发酵？
未发酵的菜籽饼直接施用会产生高热和有机酸，易**烧根**并**招引虫害**。

### 方法一：快速堆沤法（推荐，15~30天）
1. 将菜籽饼**粉碎**，颗粒越小越好
2. 按比例混合：100kg碎饼 + 200g EM发酵菌剂 + 清水（湿度50%，手握成团不滴水）
3. 堆高约1米，覆盖塑料薄膜
4. 每10天翻堆1次
5. **发酵完成标志**：颜色变黑褐色，无酸臭味，有淡淡清香味

### 方法二：密封水沤法（20~40天）
1. 碎饼与清水按 **1:1** 装入密封桶
2. 加酵母粉 + 橘皮（去臭）
3. 定期放气（胀气时拧盖放气）
4. 液体变黑褐色即可兑水使用

### 方法三：家庭简易法（1~3个月）
1. 菜籽饼捣碎，加入3~4倍种植土混合
2. 浇透水，用塑料布盖严
3. 室外存放，定期翻动
4. 无难闻味道、颜色变黑即完成

## 三、施用方法

### 基肥（播种/移栽前）
| 场景 | 用量参考 |
|------|---------|
| 蔬菜（叶菜类） | 每亩 **200~300kg** |
| 果树 | 每株 **0.5~1kg**，环状沟施 |
| 家庭菜园（每平方米） | **1~2kg** |
| 花圃/盆栽 | 盆土的 **1/10~1/5** |

**操作步骤：**
1. 将发酵好的菜籽饼均匀撒在土壤表面
2. 深耕翻入土中，深度 **20~30cm**
3. 翻土后静置 **7~10天** 再播种
4. 可与复合肥搭配使用，效果更佳

### 追肥（生长期间）
| 形态 | 方法 | 稀释比例 |
|------|------|---------|
| 固体追肥 | 穴施（距根部10~15cm） | — |
| 液体追肥 | 发酵液兑水浇灌 | **10~20倍** |
| 叶面喷施 | 发酵滤液喷叶面 | **50倍** |

**追肥频率**：每月 **1~2次**，薄肥勤施

### 施用技巧
- 瓜果类（番茄/茄子/辣椒）：膨果期穴施 + 配草木灰（3:1）
- 叶菜类：发酵液肥稀释50倍喷施，10天1次
- 豆科作物：可适当增加磷肥用量

## 六、常见问题（FAQ）

### Q1：会不会烧根？
**A**：发酵好的不会。未发酵或距根部太近会。正确操作：保持 **10cm以上距离**，与土壤充分混合。

### Q2：多久见效？
**A**：基肥约 **2~3周** 见效，追肥约 **7~10天**。有机肥肥效缓释持久，不追求速效但持续性好。

### Q3：可以和其他肥混用吗？
**A**：可以。基肥阶段与复合肥搭配效果更好；追肥阶段建议单独使用有机肥。

### Q4：会有味道吗？
**A**：发酵好的成品无异味，有淡淡清香味。若有明显臭味，说明未完全发酵。

### Q5：家庭阳台能用吗？
**A**：能用，用量减半。注意充分发酵，盆栽月季、花卉、果树均适用。

### Q6：可以保存多久？
**A**：干燥通风条件下可保存 **6~12个月**。避免受潮、生霉。

### Q7：下雨天能施肥吗？
**A**：不建议雨天施基肥，养分易流失。建议晴天施用，配合浇水。

### Q8：对土壤有什么好处？
**A**：
- 增加土壤有机质，改善团粒结构
- 提高土壤保水保肥能力
- 促土壤有益微生物繁殖
- 减少土传病害
